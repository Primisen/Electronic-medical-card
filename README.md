# Electronic medical card

I think not need comments for what this application:)

Application has two user roles: doctor and patient. 

I used the below thechnologies in this project:
* Java 11;
* Spring Framework;
* Spring MVC;
* Spring Security;
* Hibernate;
* Maven;
* Apache Tomcat;
* MySQL;
* Bootstrap 5.
